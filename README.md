## What's inside

**CAM core — included in every tier**
4-axis G-code post-processor (synced X/Y + independent U/V) · auto-nesting across parts · bounding-box collision check · pre-cut wire-safety validation · on-screen real-machine jogging · live 4-axis DRO (+ optional 5th rotary) · heater control (external / board PWM / PID) · SHAPES primitives + SVG / DXF import · material library (EPS / XPS / PU + custom) · Quick Lab calibration · controller auto-detect.

**Pro Packs — mix only the generators you need**

- 🏛️ **BUILD** — parametric domes, vaults and buildings
- ✈️ **AERO** — NACA-4 airfoils, tapered wings, spar slots, 4-axis tilted cuts
- 🌡️ **THERMAL** — pipe-shell unfolder + thermal panels with half-lap joints
- 🎨 **DECOR** — catalogue cornice / moulding profiles, a freehand **Grid Sketch** editor, and **foam lettering & signs** from any installed font

## ✨ Highlights of this release

- **New DECOR pack** — the fourth Pro Pack: a catalogue of real **cornice / moulding** profiles scaled to your dimensions, the freehand **Grid Sketch** custom-contour editor (snap grid · curved edges · inner holes · trace-over-image · saved-shape library), and a **Text** generator for foam letters and signs from any font.
