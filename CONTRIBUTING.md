# Contributing

FoamSync is proprietary, closed-source software — **source patches
are not accepted**. There are several other ways to help.

## Bug reports

Open a [GitHub issue](https://github.com/Balcore-Systems/foamsync/issues)
with:

- FoamSync version (visible in the licence badge or on the
  activation dialog).
- Windows version.
- A description of what you did and what you observed.
- A [diagnostic bundle](docs/support.md#filing-a-useful-bug-report)
  attached or referenced.

Reproducible bugs are easier to fix. If you can isolate the action
that triggers it, please describe the steps.

## Feature requests

Open a [GitHub issue](https://github.com/Balcore-Systems/foamsync/issues)
with the `enhancement` label and describe:

- The workflow you are trying to support.
- How you handle it today.
- What would change for you if FoamSync supported it natively.

Concrete operator stories help us decide what to prioritise.

## Documentation improvements

The Markdown files in this repository accept pull requests. Typos,
clarifications, and additional examples are all welcome. For larger
structural changes, open an issue first so we can discuss the layout.

## Translations

Translations of the documentation and of the in-bot interactions are
welcome via pull request. Open an issue first if you want to start a
new language so we can coordinate.
